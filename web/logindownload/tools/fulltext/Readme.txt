If the Search functionality was disabled when you installed Olate Download 
3 due to your MySQL version, you can enable it by running this file.

Copy the file into your OD3 directory (e.g. upload/) and then upload to your
website. Using your web browser, browse to the location of the page and follow
the on screen instructions.

e.g. if you uploaded the upload/ directory to /downloads, you would browse to
http://www.example.com/downloads/fulltext.php

Once you have finished, delete the file from your site.
